# RatnagiriHub

A local business directory web app for Ratnagiri, Maharashtra — helping people discover shops, hotels, restaurants, Alphonso mango vendors, and other local services in the area.

## About

RatnagiriHub is being built to connect the Ratnagiri community with trusted local businesses. Users can browse listings by category (shops, hotels, restaurants, health services, automobile, travel, and more), search, view business profiles, and add or manage their own listings.

## Tech Stack

- **React** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (styling)
- **React Router** (navigation)
- **React Query** (data fetching)
- **Firebase** (authentication)

## Status

🚧 Work in progress — core structure is set up; pages, components, and backend integration are being added incrementally.

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```
